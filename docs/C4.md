## Nível 1 — Contexto do Sistema

Mostra o FrogBets e seus usuários/sistemas externos.

```
┌─────────────────────────────────────────────────────────────────────┐
│                         FrogBets System                             │
│                                                                     │
│  ┌──────────────┐    usa    ┌─────────────────────────────────────┐ │
│  │   Usuário    │ ────────► │         FrogBets                    │ │
│  │  (apostador) │           │  Plataforma de apostas virtuais P2P │ │
│  └──────────────┘           │  para partidas de CS2               │ │
│                             └─────────────────────────────────────┘ │
│  ┌──────────────┐    usa                        │                   │
│  │    Admin     │ ──────────────────────────────┘                   │
│  │  (gerencia   │                                                    │
│  │  jogos/times)│                                                    │
│  └──────────────┘                                                    │
└─────────────────────────────────────────────────────────────────────┘
```

**Usuário (apostador):** Membro do grupo FrogEventos. Acessa via convite. Cria e cobre apostas, acompanha leaderboard, visualiza ranking de jogadores.

**Admin:** Usuário com permissão elevada. Gerencia jogos, times, jogadores, convites, resultados e estatísticas de partida.

Não há integrações com sistemas externos — a plataforma é completamente self-contained.

## Fluxos Principais

### Fluxo de Aposta P2P

```
Usuário A                    API                         Banco
    │                         │                            │
    │── POST /api/bets ───────►│                            │
    │   { marketId, option,   │── BEGIN SERIALIZABLE ─────►│
    │     amount }            │── SELECT user balance ─────►│
    │                         │◄─ balance ok ───────────────│
    │                         │── UPDATE VirtualBalance -= X│
    │                         │── UPDATE ReservedBalance += X
    │                         │── INSERT Bet (Pending) ─────►│
    │                         │── COMMIT ──────────────────►│
    │◄─ 201 Bet created ──────│                            │
    │                         │                            │
Usuário B                     │                            │
    │── POST /api/bets/{id}/cover ►│                       │
    │                         │── BEGIN SERIALIZABLE ─────►│
    │                         │── SELECT bet (Pending) ────►│
    │                         │── UPDATE Bet → Active ──────►│
    │                         │── UPDATE UserB balance ─────►│
    │                         │── COMMIT ──────────────────►│
    │◄─ 200 Bet covered ──────│                            │
```

### Fluxo de Liquidação

```
Admin                        API                         Banco
    │                         │                            │
    │── POST /api/games/{id}/results ►│                   │
    │   { marketId, winningOption }   │                   │
    │                         │── GameService.RegisterResult
    │                         │── SettlementService.Settle │
    │                         │   ├── Busca todas as Bets Active do Market
    │                         │   ├── Para cada Bet:       │
    │                         │   │   ├── Vencedor: VirtualBalance += 2X
    │                         │   │   │             ReservedBalance -= X
    │                         │   │   └── Perdedor: ReservedBalance -= X
    │                         │   └── Bet.Status = Settled │
    │◄─ 200 OK ───────────────│                            │
```

### Fluxo de Rating de Jogador

```
Admin                        API                         Banco
    │                         │                            │
    │── POST /api/map-results ►│                           │
    │   { gameId, mapNumber,  │── MapResultService ───────►│
    │     rounds }            │── INSERT MapResult ────────►│
    │◄─ 201 MapResult ────────│                            │
    │                         │                            │
    │── POST /api/players/{id}/stats ►│                   │
    │   { mapResultId, kills, │── MatchStatsService        │
    │     deaths, assists,    │   ├── Busca MapResult ─────►│
    │     totalDamage,        │   ├── RatingCalculator.Calculate(
    │     kastPercent }       │   │     kills, deaths, assists,
    │                         │   │     damage, mapResult.Rounds, kast)
    │                         │   ├── INSERT MatchStats ───►│
    │                         │   └── UPDATE CS2Player      │
    │                         │         PlayerScore += rating
    │                         │         MatchesCount += 1   │
    │◄─ 201 MatchStats ───────│                            │
```
